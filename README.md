# Run in development

1. Clone the repository

2. Run
  
  ```bash
  $ yarn install
  ```

3. Install nest/cli
    
    ```bash
    $ npm i -g @nestjs/cli
    ```

4. Up the database

    ```bash
    $ docker-compose up -d
    ```
